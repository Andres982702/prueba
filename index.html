🔧 Configuración de Firebase (una sola vez)

Para que todos los dispositivos compartan los mismos datos, necesitas conectar tu propio proyecto de Firebase gratuito. Sigue estos pasos:

  1. Ve a console.firebase.google.com e inicia sesión con tu cuenta Google
  2. Crea un proyecto nuevo (o usa uno existente)
  3. Ve a Firestore Database → Crear base de datos → Modo de prueba → Selecciona tu región
  4. Ve a Configuración del proyecto (ícono ⚙️) → General → baja hasta "Tus apps" → agrega una app Web (</>)
